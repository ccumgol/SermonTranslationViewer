# 설치 및 운영 매뉴얼

> 영문: [setup-guide.en.md](setup-guide.en.md) · 개발 트러블슈팅: [troubleshooting-log.md](troubleshooting-log.md)

이 프로그램은 **온라인(Gemini)** 과 **오프라인(로컬 모델)** 두 방식으로 실시간 자막을
만들 수 있고, 운영 중 화면에서 서로 전환할 수 있습니다.

---

## 1. 사전 준비물

| 항목 | 설명 |
|---|---|
| 컴퓨터 | macOS / Windows / Linux (Python 3.10+). **오프라인 모드는 Apple Silicon Mac 필요** |
| 오디오 인터페이스 | 믹서의 AUX/REC OUT → USB 오디오 인터페이스 |
| 인터넷 | 온라인 모드에만 필요(유선 권장). 오프라인 모드는 불필요 |
| Gemini API 키 | 온라인 모드에만 필요. https://aistudio.google.com/apikey |

> **권장**: 믹서에서 **설교자 마이크(설교 채널)만 분리된 AUX 버스**를 받으세요.
> 회중석 마이크·반주가 섞인 메인 출력은 인식률이 떨어집니다.

## 2. 설치

```bash
python3 -m venv .venv
source .venv/bin/activate          # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

`sounddevice` 는 PortAudio 가 필요합니다.
- macOS: `brew install portaudio`
- Ubuntu: `sudo apt install libportaudio2`

## 3. API 키 설정 (온라인 모드만)

> 오프라인 모드만 쓸 거면 이 단계를 건너뛰고 5·6장으로 가세요.

### 3-1. 반드시 개인 Google 계정으로

모델 `gemini-3.5-live-translate-preview` 는 **Gemini Developer API(API 키 방식) 전용**이며,
**조직(Workspace) 계정은 보안정책으로 API 키 발급이 막혀** 있습니다. **개인 Gmail 계정**으로
발급하세요. (자세한 이유는 트러블슈팅 문서 참고)

### 3-2. 결제(Billing) 연결

이 모델은 무료 티어로 안 됩니다. AI Studio → **결제**에서 프로젝트를 **Tier 1(유료)**로
설정하세요. (카드 등록은 본인이 직접)

### 3-3. 키 발급 → `.env` 저장

1. AI Studio → **API 키** → **API 키 만들기** → 결제 연결된 프로젝트 선택 → `AIza...` 복사
2. 저장:
   ```bash
   cp .env.example .env
   ```
   `.env` 를 열어 `GEMINI_API_KEY=...` 입력. **이 파일은 절대 공유·커밋 금지**(git 자동 제외).

## 4. 오디오 입력 점검

```bash
python server/audio_input.py --list                       # 장치 목록
python server/audio_input.py --monitor --device "장치이름"  # 말할 때 막대가 움직이는지
```
확인한 장치 번호(또는 이름)를 `.env` 의 `AUDIO_INPUT_DEVICE` 에 입력 (예: `2`).

---

## 5. 실행

```bash
./start.sh                 # mac/linux (가상환경 자동 활성화 + 서버 실행)
start.bat  또는  start.ps1  # Windows
```
- 실행하면 **운영자 화면이 자동으로 브라우저에 열립니다** (끄려면 `OPEN_BROWSER=0 ./start.sh`)
- 시작 백엔드는 `.env` 의 `BACKEND=gemini|local` 로 결정 (실행 중 화면에서 전환 가능)
- 종료: `Ctrl+C`

| 화면 | 주소 | 용도 |
|---|---|---|
| 송출 자막 | http://localhost:8000/ | 프로젝터 / OBS 브라우저 소스 |
| 운영자 | http://localhost:8000/operator | 모든 제어·모니터링 |
| 모바일 | http://localhost:8000/m | 교인 폰에서 언어 선택해 보기 |

## 6. 운영자 대시보드 사용법

예배 중 모든 조작은 **운영자 화면**에서 합니다. 상단 우측 **EN/한글** 버튼으로 UI 언어 전환.

### 6-1. 방송 시작 / 종료
- **▶ 방송 시작 / ■ 방송 종료** — 번역 파이프라인 자체를 켜고 끕니다.
- **종료 시 온라인 비용·연산이 실제로 멈춥니다**(언어 구성은 유지). 예배 전후에 사용.
- 참고: `.env` 에 `AUTO_START=0` 이면 **방송 종료 상태로 시작**(비용 0)해 필요할 때 켤 수 있음.

### 6-2. 백엔드 토글 (온라인 ↔ 오프라인)
- 상단 **백엔드 배지 클릭** → 온라인(Gemini)↔오프라인(로컬) 실시간 전환. 언어 구성 유지.
- 오프라인 전환 시 모델 로딩에 ~10초. **인터넷 끊기면 클릭 한 번으로 오프라인 폴백**.

### 6-3. 오디오 입력 소스 전환
- 드롭다운에서 장치 선택 → **즉시 전환, 세션 안 끊김**(입력 스트림만 교체).
- 예배 중 무선 마이크가 끊기면 내장 마이크로 1클릭 전환. 🔄 로 목록 새로고침.

### 6-4. 번역 언어 (1~3개 동시)
- 슬롯 3개로 목표 언어 1~3개 선택, 각 언어 **글자색** 지정.
- 송출 화면이 언어 수만큼 행으로 나뉘어 각 색으로 표시. **⚠️ 언어 N개 = 비용 N배**(온라인).

### 6-5. 설교 원고 (전사 정확도 ↑, 권장)
- "설교 원고" 칸에 원고를 붙여넣고 **원고 적용**.
- 원고의 **고유명사를 뽑아 전사를 자동 교정**합니다(예: "티브리어"→"히브리어" → 번역도 정확).
- 로컬 모드에선 **번역 전에** 교정되어 번역까지 바로잡힙니다. 원고는 재시작 후에도 유지.

### 6-6. 자막 스타일 · 출력 영역
- **폰트 크기 / 배경색 / 안여백** 실시간 조절.
- **자막 영역**: 전체 화면 / **하단 밴드** / **상단 밴드**.
- **최대 줄 수**: 제한 없음 / 1 / 2 / 3 (글자 크기와 무관하게 줄 고정).
- 활용: 단일 언어일 때 **하단 밴드 + 2줄** → 뒤에 비디오/PPT 깔고 하단에만 자막.
  배경을 **초록**으로 하고 OBS **색상 키**를 걸면 자막만 남기고 합성 가능.

### 6-7. 화면 제어
| 버튼 | 동작 |
|---|---|
| **🖥 송출 창 열기** | 송출 화면을 별도 창으로. 그 창에서 클릭/F 로 전체화면(순수 검정) |
| **↺ 화면 리셋** | 누적 자막 즉시 비움 |
| **⏹ 송출 가리기 / ▶ 송출 재개** | 송출 화면만 가림(번역은 계속) |

### 6-8. 모바일 주소 / QR (교인 폰 안내)
- **📱 모바일 주소 / QR 추출** → 접속 주소와 QR 표시(서버가 LAN IP 자동 검출, 오프라인 동작).
- QR 클릭/🔍 버튼 → **별도 창**으로 큰 QR (운영자 화면·송출을 계속 쓰면서 열어둘 수 있음).
- 교인은 같은 와이파이에서 QR 스캔 → `/m` 에서 자기 언어 선택.

### 6-9. 사용량 배지
- 상단에 **경과시간·예상비용**(온라인) 표시. **무발화 5분 경과 시 경고** — 끄는 걸 잊어
  비용이 쌓이는 것을 방지.

> 송출 화면을 새로 열거나 OBS 를 재시작해도 현재 상태(스타일·언어·송출·영역)가 자동 동기화됩니다.

## 7. OBS / 프로젝터 송출

- **OBS**: 소스 추가 → 브라우저 → `http://localhost:8000/`. 배경을 초록으로 두고 **색상 키**
  필터를 걸면 자막만 합성됩니다.
- **프로젝터 직접**: 송출 창을 프로젝터 화면으로 옮긴 뒤 **클릭해 전체화면**(주소창·타이틀 제거).
- **주소창만 없는 창**이 필요하면 Chrome 에서 그 페이지를 **앱으로 설치**(PWA)하면 됩니다.

## 8. 오프라인(로컬) 모드 설치

인터넷·API 비용 없이 동작. **Apple Silicon Mac(M1 이상)** 필요.

```bash
pip install -r requirements-offline.txt     # mlx-qwen3-asr
brew install ollama ffmpeg                   # 이미 있으면 생략
ollama pull translategemma:4b                # 번역 모델(빠름·기본, ~3GB)
#   더 정확한 12B 를 원하면: ollama pull translategemma:12b

BACKEND=local ./start.sh
```

구성: **Qwen3-ASR**(한국어 STT, 침묵으로 발화 구간을 잘라 전사) + **TranslateGemma**(Ollama 번역).
한 번 전사한 한국어를 여러 언어로 번역합니다.

튜닝(`.env`, 선택):
| 변수 | 설명 |
|---|---|
| `MT_MODEL` | `translategemma:4b`(빠름·기본) / `translategemma:12b`(정확·느림) |
| `STT_MODEL` | `Qwen/Qwen3-ASR-0.6B`(빠름·권장) / `1.7B` |
| `STT_MIN_SILENCE_SEC` | 발화를 끊는 침묵 길이(기본 0.9). **높이면 문장 중간 마침표↓·지연↑**(1.2~1.5) |
| `STT_MAX_SEGMENT_SEC` | 쉼 없이 길 때 강제 분할(기본 8초). 낮추면 더 빨리 전사 |
| `STT_SILENCE_RMS` | 침묵 음량 임계값. 말해도 안 뜨면 낮추고(0.008), 잡음 반응하면 높임(0.025) |

**운영 팁**: 로컬은 **또렷한 단문 + 문장 끝 짧은 멈춤**일 때 가장 정확합니다. 반복되는 특정
오인식은 `data/glossary/glossary.txt` 의 `fix:` 규칙으로 확실히 고정하세요
(예: `fix: 티브리어 => 히브리어`).

## 9. 보안 (LAN 노출 주의)

기본값 `WS_HOST=0.0.0.0` 은 같은 LAN 의 폰/PC 에서 접속 가능(모바일 자막에 필요)하지만,
`/operator` 명령에 **인증이 없으면** 같은 네트워크의 누구나 설정을 바꿀 수 있습니다.

- `.env` 에 **`OPERATOR_TOKEN=<긴 무작위 문자열>`** 설정 → `http://…/operator?token=…` 로만 조작 가능
- 이 PC 에서만 쓸 거면 `WS_HOST=127.0.0.1`
- 송출(`/`)·모바일(`/m`)은 읽기 전용이라 토큰 없이 봐도 안전

## 10. 예배 전 체크리스트

- [ ] 믹서 → 인터페이스 → 컴퓨터 입력 레벨 정상 (`--monitor`)
- [ ] 무선 마이크 배터리 / (온라인이면) 유선 인터넷
- [ ] `./start.sh` 실행 → 운영자 화면 "연결됨", 원하는 백엔드 배지 확인
- [ ] 설교 원고가 있으면 붙여넣고 **원고 적용**
- [ ] 시험 발화 시 1~3초 내 자막 갱신
- [ ] **▶ 방송 시작** 눌러 두기 (또는 AUTO_START)
- [ ] 필요하면 **📱 QR 추출**해서 교인 안내

## 11. 장애 시 대응

| 증상 | 조치 |
|---|---|
| 자막이 안 뜸 | 운영자 화면 "연결됨" 확인 → **▶ 방송 시작** 눌렀는지 → 서버 로그 확인 |
| 자막이 멈춤 | 서버 자동 재연결(2초). 지속 시 재시작 |
| 소스가 잘못됨 | **오디오 입력 소스** 드롭다운으로 전환 (세션 안 끊김) |
| 고유명사 오인식 | **설교 원고** 붙여넣기 · 용어집 `fix:` 규칙 |
| 인터넷 끊김 | **백엔드 배지 클릭 → 오프라인 폴백** (로컬 설치돼 있으면) |
| 문장 중간 마침표(로컬) | `STT_MIN_SILENCE_SEC=1.2` 로 올리기 |
| (로컬) 자막 안 뜸 | 입력 장치 확인 + `STT_SILENCE_RMS=0.008` |
| 폰에서 QR 접속 안 됨 | 교회 와이파이 게스트 격리(client isolation) 여부 확인 |
| 최후 수단 | **⏹ 송출 가리기** 또는 **■ 방송 종료** — 음성 송출은 계속 |

## 12. 비용

- **온라인(Gemini)**: 약 **$0.037/분 × 언어 수**. 40분 설교 1언어 ≈ $1.5 미만, 3언어 ≈ $4.5.
  청구는 [AI Studio 결제](https://ai.google.dev/gemini-api/docs/billing). 예배 후 **■ 방송 종료** 잊지 마세요.
- **오프라인(로컬)**: **비용 0** (전기·하드웨어만). 인터넷 불필요.
